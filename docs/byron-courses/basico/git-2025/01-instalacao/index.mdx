# Instale as ferramentas necessárias

Antes de começarmos, precisamos que nosso ambiente de trabalho esteja com as ferramentas adequadas para utilzarmos nesta parte da capacitação. Você pode clicar [aqui](../../git/ambiente/) para acessar o passo a passo da versão anterior desta capacitação, ou navegar até a aba "Configurando o ambiente".
